# 📋 病历管家・报告自动记录 Skill

肿瘤 / 慢病患者治疗数据自动化管理工具 | 基于 CodeBuddy 低代码 Skill 开发

针对肿瘤、慢病患者需长期频繁记录检查 / 检验数据的刚需，解决手动抄录繁琐易错、数据零散不规范的痛点，实现 **"发图即录"** 全自动数据管理。从 5 分钟手动录入 → 30 秒自动归档，效率提升 90%+。

## ✨ 核心功能

### 📸 多类型报告识别

已支持主流医疗报告智能识别：
- 血常规报告
- 血脂检测
- 肿瘤标志物（CEA/CA19-9/CA125 等）
- CT/MRI 影像报告（原发灶 / 转移灶追踪）

### 🔬 指标自动提取

- OCR 智能识别报告内容
- 自动提取关键指标数值
- 统一单位换算与格式标准化

### ⚠️ 异常状态评估

基于临床参考实现 0–IV 级分级判断：

| 级别 | 图标 | 含义 | 处理建议 |
|------|------|------|----------|
| 0级 | (无标记) | 正常范围 | 无需处理 |
| I级 | ⚠️ | 轻度异常 | 观察监测 |
| II级 | 🟠 | 中度异常 | 建议就诊 |
| III级 | 🔴 | 重度异常 | 紧急就医 |
| IV级 | 🚨 | 极重度/危急值 | 立即就医 |

> 正常指标不额外标记，仅展示数值，界面更清爽。

### 💊 智能处理建议

- 分级处理原则（参考出院医嘱）
- 症状观察与自我监测指导
- 饮食、活动等生活护理建议

### 📊 台账自动规整

- 数据格式统一规范化
- 自动录入飞书表格
- 历史数据归档，便于趋势对比与复诊展示

## ⚠️ 危急值警示（最高优先级）

出现以下情况**立即联系医生或急诊**：

| 危急值 | 风险 |
|--------|------|
| ANC < 0.5×10⁹/L | 高感染风险 |
| PLT < 20×10⁹/L | 自发性出血风险 |
| Hb < 60 g/L | 重度贫血风险 |
| 体温 ≥ 38.0℃ + 低 ANC | 发热性中性粒细胞减少症 |

> 处理原则：发现危急值立即就医，不等待！
